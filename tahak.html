PŘEHLED KLÁVESOVÝCH ZKRATEK PRO BLENDER

POHYB V PROSTŘEDÍ

  • MMB* (střední tlačítko myši) - otočení pohledu
  • Kolečko myši - přiblížení / oddálení
  • Shift + MMB* - posun pohledu

REŽIMY PRÁCE

  • Tab - přepnutí mezi Object Mode a Edit Mode

PŘIDÁVÁNÍ OBJEKTŮ

  • Shift + A - Add Menu (přidání krychle, koule, roviny…)

ZÁKLADNÍ TRANSFORMACE

  • G - posun (Move)
  • S - zvětšení / zmenšení (Scale)
  • R - otočení (Rotate)
  • X, Y, Z- omezení pohybu na osu (např. G + X)
  • E - extrude (vytáhnout plochu)
  • I - inset (vytvořit vnitřní plochu)

  • RMB* - nabídka akcí pro označené plochy/vrcholy/hrany
  • Subdivide - pravé tlačítko → Subdivide

PRÁCE SE SCÉNOU

  • Delete nebo X - smazat
  • Shift + D - duplikovat objekt
  • Ctrl + Z - krok zpět
  • Ctrl + Shift + Z - krok dopředu
  • Ctrl + S - uložit

*

  • RMB = Pravé tlačítko na myši.
  • MMB = Prostřední tlačítko myši (kliknutí na kolečko).
  • LMB = Levé tlačítko na myši.
